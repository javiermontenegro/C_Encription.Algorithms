* [SHA-1](https://en.wikipedia.org/wiki/SHA-1) 
* [SHA-256](https://en.wikipedia.org/wiki/SHA-2) 


### Compile the program

Build the program using gcc.
```
$ gcc source_script.c main.c -o source_script 
```
## Running the program 
```
$ ./executable_script
```
